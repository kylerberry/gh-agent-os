# Knowledge base

This directory is a starter knowledge base for the GitHub Agent Factory. It lets humans and sandboxed agents find operational rules, prior decisions, and canonical task context without treating a large pile of documents as prompt context.

## How to use it

- Start at [`wiki/index.md`](wiki/index.md) for a task-oriented map.
- Read a wiki page first; it should summarize the current state and link to canonical source material.
- Use `raw/` only when exact requirements, an ADR rationale, or source wording is needed.
- Record substantive new decisions as an ADR, feature behavior as a spec, and external findings as research. Update the corresponding wiki page and index.

Read [`AGENTS.md`](AGENTS.md) for the agent lookup and maintenance protocol.
