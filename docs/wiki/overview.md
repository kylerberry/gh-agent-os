# GitHub Agent Factory Overview

This repository packages a human-governed GitHub Actions workflow for coding agents. GitHub Issues provide the task queue; optional GitHub Projects visualize lifecycle state. Agents can research, create same-repository implementation branches and PRs, and address material review feedback. Humans authorize implementation and merge.

The canonical workflow context is [`docs/raw/context/github-agent-factory-context.md`](../raw/context/github-agent-factory-context.md). See the architecture and operations pages in the [index](index.md) for the runnable system.
